# ta191
Low Noise, improved Moving-Coil Phono Pre-Amp

## Purpose
An optional _previous_ stage for **MM Phono preamps** like **TuchAudio 101b**, allowing the use of **Moving Coil** cartridges. _With suitable wiring_ it could be used with other preamps as well, like the `103` board.

A _built-in relay_ allows the **same input connector** to be used with both **MC** (thru amplification) and **MM cartridges** (_bypassing_ it straight to the MM Phono stage)
